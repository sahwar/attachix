WebDAV for Android

Android can access Attachix

Tested with Android 2.2.

  1. Download WebDAV Navigator Lite (free).
  2. Open WebDAV Navigator Lite.
  3. Press Menu and then Add Server.
  4. Set "Name" as "Attachix".
  5. Set "Server URL" as "http://attachix.de/" or the shared with you URL (something like http://ab4chQf.k.attachix.de/).
  6. Set "Username" as your CloudMe username.
  7. Set "Password" as your CloudMe password.
  8. Click Add Server.
  9. Select "CloudMe" in the list of servers.
