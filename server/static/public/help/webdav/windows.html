Need help configuring NetDrive? Read below.

You can configure NetDrive to use Attachix by following these instructions:

  1. Click New Site (bottom left button)
  2. Site Name = Attachix
  3. Site IP or URL = http://attachix.de/ or the shared with you URL (something like http://ab4chQf.k.attachix.de/)
  4. Server Type = WebDAV
  5. Drive = Z (or any other)
  6. Account = Your Attachix account username (if someone shared attachix URL to you then don't enter username)
  7. Password = Your Attachix account password (if someone shared attachix URL to you then don't enter password)
  8. Check the box - Automatic login when NetDrive starts so that you don't have to reenter everything.
  9. Click Connect and you will then be connected to your attchix disk, which contains all your files.
