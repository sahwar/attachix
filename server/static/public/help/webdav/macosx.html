WebDAV for Mac OS

Mac OS has built in support for WebDAV

  1. First open Go in the main menu, while no application is focused and select Connect to Server...
  2. Type in the full URL to the service: "http://attachix.de" or the shared with you URL (something like http://ab4chQf.k.attachix.de/) and click Connect
  3. Type in your username (in lowercase) and password in the authentication dialog and click OK
