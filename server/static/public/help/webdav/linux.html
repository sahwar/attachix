WebDAV for Linux

Linux has built in support for WebDAV

  1. First open your File Explorer (Nautilus or Konqueror).
  2. Open File in the menubar and select Connect to Server...
  3. In the dialog that appears select WebDAV (HTTP) as Service type
  4. Set Server as "attachix.de" or the shared with you URL (something like ab4chQf.k.attachix.de)
  5. Click Connect
  6. Type in your username (in lowercase) and password in the authentication dialog that appears and click Connect
